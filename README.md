## Getting Started

### How to run pokedex server
```bash
cd server
yarn install
yarn start
```
The server will be started on port 3030 (http://localhost:3030)

### How to run pokedex web
```bash
cd web
yarn install
yarn start
```

The web page will be started on port 3000 (http://localhost:3000)
